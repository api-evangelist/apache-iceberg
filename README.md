# Apache Iceberg (apache-iceberg)
Apache Iceberg is an open table format for large analytic datasets that provides ACID transactions, schema evolution, hidden partitioning, and time travel. It works with Spark, Flink, Hive, Presto, and Trino.

## APIs

### Apache Iceberg
Iceberg provides a Java API for table operations, schema management, and partition management, a REST Catalog API for table discovery and metadata management, and integration APIs for Spark, Flink, Hive, and Trino.
